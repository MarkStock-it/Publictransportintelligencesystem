- Real-time GPS simulation
- Color-coded occupancy (Green/Yellow/Red)

- Click jeeps → Occupancy details
- Night safety mode

### Route Intelligence  
- System-wide analytics
- Per-route performance
- Reliability scores (85-97%)
- Peak hour chips
- Predictive insights

### LGU Dashboard
- Congestion heatmaps
- 24-hour demand patterns
- Route imbalance alerts
- Policy recommendations
- Wait time analysis

### Low Bandwidth
- Text-only interface
- ~50KB data usage
- Accessible design
